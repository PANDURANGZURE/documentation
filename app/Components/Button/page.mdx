## Technical Summary of Output

| Feature | CSS Class Applied | Expected Result |
| :--- | :--- | :--- |
| **Primary** | `.primary` | Solid black background, white text. |
| **Outline** | `.outline` | Transparent background, 1px black border. |
| **Ghost** | `.ghost` | No border/background until hover. |
| **Loading** | `N/A` | Text changes to "Loading..." and button is disabled. |
| **Link** | `N/A` | Renders as `<a>` tag with `href` instead of `<button>`. |
